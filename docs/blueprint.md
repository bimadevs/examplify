# **App Name**: Examplify

## Core Features:

- Guru Data Input: Form to input guru data (name, ID, subject).
- Siswa Data Input: Form to input siswa data (name, ID, class).
- Soal Data Input: Form to input soal data (question, options).
- Exam UI: Siswa view of exam questions, options and submit button
- Hasil Analysis: AI tool analyzes student performance and provides insights.

## Style Guidelines:

- Primary color: Strong Blue (#4285F4) to invoke a sense of trust and focus. 
- Background color: Light blue (#E3F2FD) as a soft, neutral backdrop that doesn't distract from the content.
- Accent color: Analogous sky blue (#64B5F6) to guide user interactions and indicate progress.
- Body and headline font: 'Inter' (sans-serif) for clear readability and a modern, neutral appearance.
- Code Font: 'Source Code Pro' (monospace) is used to represent source code within the webpage. Note: currently only Google Fonts are supported.
- Simple, clear icons for navigation and actions within the platform.
- Clean, structured layout with ample white space to avoid cognitive overload.
